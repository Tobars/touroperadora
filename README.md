# Tour Operadora - El Encanto Travel

Web application for a travel agency that offers tour packages and travel services.

## Project Structure

```
touroperadora/
├── web/
│   ├── css/
│   │   └── styles.css          # All CSS styles
│   ├── js/
│   │   └── main.js             # JavaScript functionality
│   ├── assets/
│   │   └── images/             # Images and media files
│   ├── index.html              # Main page
│   └── META-INF/
├── src/
│   └── conf/
├── build/
├── nbproject/                  # NetBeans configuration
└── README.md
```

## Technologies Used

- **HTML5**: Semantic markup
- **CSS3**: Custom styles with CSS variables
- **JavaScript (ES6+)**: Interactive functionality
- **Bootstrap 5.3**: Responsive framework
- **Font Awesome 6.4**: Icons

## File Organization

### CSS (`web/css/styles.css`)
Contains all custom styles including:
- CSS variables for theme colors
- Navbar styles
- Hero section with animations
- Tour cards
- Testimonials
- Footer
- Responsive design (media queries)

### JavaScript (`web/js/main.js`)
Contains:
- Intersection Observer for scroll animations
- Dynamic card animations

### Assets (`web/assets/`)
Directory for static resources:
- `images/`: Store images here

## Features

- **Responsive Design**: Works on all device sizes
- **Scroll Animations**: Cards animate when entering viewport
- **Modern UI**: Clean and professional design
- **Tour Packages**: Showcase different travel destinations
- **Testimonials**: Customer reviews section
- **Contact Information**: Footer with social links

## Development

This is a NetBeans Java Web project. To run:

1. Open project in NetBeans IDE
2. Configure your server (Tomcat, GlassFish, etc.)
3. Build and deploy the project
4. Access via browser at configured server URL

## Browser Compatibility

- Chrome (latest)
- Firefox (latest)
- Safari (latest)
- Edge (latest)
